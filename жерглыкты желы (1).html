Қарапайым жергілікті желіні (LAN) орнату — HTML нұсқаулық

Бұл бетте: желі архитектурасы, IP конфигурациясы мысалдары (статикалық және DHCP), Windows/Linux/Mac командалары, файл бөлісу және негізгі тексеру қадамдары көрсетілген. Оны оқу арқылы екі немесе бірнеше компьютерден тұратын қарапайым жергілікті желіні тез баптай аласыз.

1. Қандай жабдық қажет?

  • Маршрутизатор (router) немесе коммутатор (switch) — егер бірнеше құрылғы қосылса.
  • Ethernet кабельдері (CAT5e/6) немесе Wi-Fi роутер арқылы сымсыз қосылу.
  • Компьютер(лар), ноутбук(тар) — Windows, Linux немесе Mac.

2. Желінің қарапайым схемасы

Маршрутизатор (DHCP) —> Switch —> Компьютер1, Компьютер2, ...

3. IP конфигурациясы — негізгі түсінік

Ең жиі қолданылатын локальды желі диапазоны:

  • 192.168.0.0/24 (мысалы: 192.168.0.1 — маршрутизатор)
  • 192.168.1.0/24 (мысалы: 192.168.1.1 — маршрутизатор)
  • 10.0.0.0/24 (мысалы: 10.0.0.1 — маршрутизатор)

Subnet mask: әдетте 255.255.255.0 (/24)

Ескерту: егер маршрутизаторда DHCP қосулы болса, құрылғылар автоматты түрде IP алады. Қолмен (статикалық) бергенде әр құрылғыға бірегей IP орнатыңыз.

4. Мысал — статикалық IP орнату

Windows (GUI нұсқасы)

  1. Control Panel → Network and Internet → Network and Sharing Center → Change adapter settings.
  2. Таңдап, Properties → Internet Protocol Version 4 (TCP/IPv4) → Properties.
  3. "Use the following IP address": IP: 192.168.1.10, Subnet mask: 255.255.255.0, Default gateway: 192.168.1.1.
  4. DNS: роутердің IP немесе 8.8.8.8 және 8.8.4.4.

Linux (terminal)

Убунту/дебиан үлгісі (уақытша командалық):

sudo ip addr add 192.168.1.11/24 dev eth0
sudo ip route add default via 192.168.1.1
sudo bash -c 'echo "nameserver 8.8.8.8" > /etc/resolv.conf'
